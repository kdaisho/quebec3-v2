こんちは日本人の皆さん。

## 日本ヤバい。

ソース：兵庫 NEWS WEB（[不正プログラム書き込み疑い補導](https://www3.nhk.or.jp/lnews/kobe/20190304/2020003239.html '兵庫 NEWS WEB')）

> クリックすると同じ画面が表示され、消えなくなる不正なプログラムのアドレスをインターネットの掲示板に書き込んだとして、１３歳の女子中学生が兵庫県警に補導されました。 補導されたのは、愛知県刈谷市に住む中学１年生の１３歳の女子生徒で、警察によりますと、インターネットの掲示板に、不正なプログラムのアドレスを書き込んだ疑いがもたれています。 このプログラムでは、クリックすると、画面の真ん中に「何回閉じても無駄ですよ〜」という文字や、顔文字などが表示され続けるよう設定されているということです。 書き込みを見つけた警察が、インターネットの接続記録を調べ、４日、補導しました。 また、同じアドレスを別の掲示板に書き込んだ疑いで、山口県の３９歳の無職の男と鹿児島県の４７歳の建設作業員の男のそれぞれの自宅も捜索しました。 今後、書類送検する方針です。 警察によりますと、３人に面識はなく、それぞれネット上の別の場所にあったものをコピーして貼り付けていたということです。 調べに対し、いずれも容疑を認め、女子生徒は、「以前、自分が困ったことがあり、誰かがクリックすれば、おもしろいと思った」と話しているということです。

「3 人に面識はなく〜」って。あるか、んなもん。

Web という名の大海原の中には、色々な人々が色々なものを好き勝手に公開しててねぇ、全てが役に立つ Web サイトばかりじゃないの。サイトを開いた途端、イタズラメッセージなんかを表示し続ける悪質なものなんかもあったりするのよ。

社会が皆良い人ばかりで構成されているわけじゃないのと全く同じ原理でさ、悪い奴に作られた悪いページも探せばどっかにあるもんなの。そういったのを全て含めて Web なんだよ。一般社会と一緒なの。

上の記事を読んで思うことはさ、まぁ「無限にメッセージが表示されるサイト」のリンクをネット掲示板に貼った人たちの行為は悪いよ。それにしても補導や書類送検って兵庫県警マジ？そいつらはそのサイトを作った本人ですらないんだぞ。

リンク貼っただけだぞ？

せめてそのサイト管理してる奴を逮捕しろよ兵庫県警。ほんと何してんだか。

そもそも「不正なプログラム」っつっても、たった 3 行の JavaScript で書かれた for ループじゃねぇか。こんなのブラウザ閉じればそれで終わりじゃん。兵庫ではブラウザの概念すら無い奴がサイバー犯罪対策課の警棒握ってんのか。

それにしても「悪質サイトのリンク貼り付け」っていうピンポンダッシュと同程度の行為を「サイバー犯罪」ってドヤ顔で叫んじゃう日本のマスコミのリテラシーの低さは爆笑もの。片腹痛い 🤣 全員小学校からやり直して来い。

あーあ、日本じゃリンク貼るだけでハッカー扱いされるのかよ。いいなぁ日本。日本行ったら無双できるじゃん。
