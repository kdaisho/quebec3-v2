日本の皆さんこんにちは。ようやくこの web サイトがなんとか形になってきたんで、いい加減最初の日記でも書くかなと思ったのでこうして書いている。しかしながら久しぶりに一から web サイトつくるとなると結構手間かかるもんだ。仕事の合間をぬってチョコチョコ作り上げてきたんで結局 3 週間くらいかかってしまったんだ。それでも実装したい機能が 100%実装されたわけじゃない…

なんかこう…もっとインタラクティブ（マウスでいじったらリアルタイムに面白い反応が返ってくるような感触）にする計画なんで、これからもブログ更新しつつ時間を見つけては機能も盛り込んでいこうと思う。
まぁこんなことせずとも WordPress で書いちゃえば手っ取り早かったんだけど、なんか遅いから好きになれないんだ。

今日は日曜日、掃除機でもかけるか。
